1. Objet et champ d'application

Les présentes CGU/CGV régissent l'utilisation du service FoncierMCP, édité par Kayzen Lyon, accessible à l'adresse fonciermcp.fr.

Toute utilisation implique l'acceptation pleine et entière des présentes conditions.

2. Définitions

  • Service : outils FoncierMCP via l'API MCP ou la démo web
  • Abonné : personne ayant souscrit un abonnement payant
  • Clé API : identifiant unique d'accès au service
  • Requête : chaque appel à un outil MCP consommant du quota

3. Accès au service

La démo est accessible gratuitement, sans inscription, à usage personnel. L'accès API nécessite un abonnement et une clé API valide. La clé API est personnelle et ne peut être partagée.

4. Abonnements et tarifs

Offres disponibles (prix HT) :

  • Starter : 49€/mois (39€ annuel) — 500 requêtes, 1 clé API
  • Pro : 149€/mois (119€ annuel) — 3 000 requêtes, 5 clés API
  • Cabinet : 490€/mois (392€ annuel) — illimité

Les abonnements sont mensuels à renouvellement tacite. Tarifs modifiables avec préavis de 30 jours.

5. Paiement

Paiement exclusivement via Stripe Inc.. Kayzen Lyon ne stocke aucune donnée bancaire. Accès suspendu après 7 jours de délai de grâce en cas d'échec.

6. Résiliation

Résiliation possible à tout moment via Stripe ou par email. Effet à la fin de la période en cours, sans remboursement prorata. Le droit de rétractation 14 jours ne s'applique pas aux services numériques déjà activés (art. L.221-28 Code conso.).

7. Utilisation des données

FoncierMCP interroge des APIs publiques (DVF, Cadastre, PLU, Géorisques, DPE). Interdictions : bases de données commerciales à revendre, scraping massif, utilisation contraire à la législation.

8. Responsabilités

Objectif 99,9% uptime pour Cabinet. Kayzen Lyon n'est pas responsable de l'inexactitude des données sources tierces. L'utilisateur utilise les données sous sa seule responsabilité professionnelle.

9. Propriété intellectuelle

Code source et interfaces : propriété exclusive de Kayzen Lyon. L'abonné conserve la propriété de ses analyses.

10. Droit applicable

Droit français. Tribunaux de Lyon compétents. Contact : contact@fonciermcp.fr

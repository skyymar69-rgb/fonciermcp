// Les fonctionnalités

6 sources officielles.
1 seule requête.

Chaque outil interroge une API officielle française en temps réel. Aucune donnée n'est stockée.

🗺 01 / Cadastre

Référence cadastrale officielle

La référence cadastrale identifie de manière unique chaque parcelle de France. Lex Foncier interroge l'API Carto IGN pour obtenir ces données à partir d'une simple adresse.

Source : API Carto — IGN · Licence Ouverte v2.0

Exemple
Référence69123 AB 0287
Surface parcelle342 m²
CommuneLyon 3e
💶 02 / DVF

Prix de vente réels certifiés

La base DVF recense toutes les transactions depuis 2014. Certifiées DGFiP, non manipulables. Médiane, moyenne, fourchette sur rayon paramétrable.

Source : DVF — DGFiP / data.gouv.fr · Mise à jour trimestrielle

Exemple
Prix médian5 840 €/m²
Min / Max4 200 – 7 100 €/m²
Transactions12 ventes · rayon 500m
📐 03 / PLU

Règles d'urbanisme

Zone PLU, constructibilité, lien vers le règlement PDF. Source : Géoportail de l'Urbanisme (DGALN).

Source : Géoportail Urbanisme — IGN / DGALN

Exemple
ZoneUA — Constructible
DocumentPLU Lyon 2020
Règlement↗ PDF disponible
⚠️ 04 / Géorisques

Risques naturels et technologiques

Inondation, séisme, mouvement de terrain, radon — l'équivalent numérique de l'ERP notarial via l'API BRGM.

Source : Géorisques — BRGM / Ministère Transition Écologique

Exemple
InondationCrue lente — Zone B
SéismeZone 2 — Faible
✓ Aucun risque technologique
🏠 05 / DPE

Performance énergétique du parc

Distribution des étiquettes DPE (A à G) dans la commune. Stratégique pour identifier les passoires thermiques et anticiper la loi Climat.

Source : Base nationale DPE — ADEME · 10+ millions de DPE

Exemple
A+B43 logements (17%)
C+D140 logements (57%)
F+G23 passoires (9%)

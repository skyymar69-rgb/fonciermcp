❓ FAQ

Questions fréquentes

Réponses directes sur les données officielles utilisées par Lex Foncier.

Qu'est-ce que le cadastre IGN et à quoi sert-il ? +

Le cadastre est le registre officiel qui identifie toutes les parcelles de France. L'API Carto IGN fournit la référence parcellaire (section + numéro), la surface exacte en m² et le code INSEE. Lex Foncier la requête automatiquement à partir d'une adresse postale, via la géolocalisation BAN.

Qu'est-ce que le DVF et comment est calculé le prix au m² ? +

Le DVF (Demandes de Valeurs Foncières) recense toutes les transactions immobilières certifiées par la DGFiP depuis 2014. Lex Foncier calcule la médiane des prix au m² (plus robuste que la moyenne) sur un rayon paramétrable. Source : DGFiP / data.gouv.fr · Licence Ouverte Etalab v2.0

Qu'est-ce que l'ERP numérique fourni par Lex Foncier ? +

L'État des Risques et des Pollutions (ERP) est obligatoire pour toute transaction immobilière. Lex Foncier génère l'équivalent numérique via l'API Géorisques BRGM : inondations, séismes, mouvements de terrain, radon, retrait-gonflement des argiles. Source : BRGM / Ministère Transition Écologique

Combien de DPE sont disponibles dans la base ADEME ? +

La base nationale des DPE de l'ADEME contient plus de 10 millions de diagnostics géolocalisés depuis juillet 2021. Pour le seul département du Rhône (69), 353 820 DPE sont disponibles. Lex Foncier les requête par coordonnées GPS dans un rayon de 300m autour de l'adresse analysée.

Les données sont-elles à jour et fiables ? +

Oui. Toutes les données sont interrogées en temps réel via les APIs officielles françaises (IGN, DGFiP, BRGM, ADEME, INSEE). Elles sont sous Licences Ouvertes Etalab v2.0. Lex Foncier ne stocke aucune donnée — chaque requête interroge directement la source officielle.

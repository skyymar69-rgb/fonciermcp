⚡ Démarrage rapide — Claude Desktop

1
Obtenez votre clé API
Abonnez-vous sur fonciermcp.fr/tarifs. Clé envoyée par email en <2 minutes.
2
Ouvrez la config Claude Desktop
Menu Claude → Settings → Developer → Edit Config. S'ouvre le fichier claude_desktop_config.json.
3
Ajoutez FoncierMCP
Copiez ce bloc dans la section mcpServers :
json
{
  "mcpServers": {
    "fonciermcp": {
      "command": "npx",
      "args": ["-y", "@fonciermcp/client"],
      "env": {
        "FONCIERMCP_API_KEY": "votre_cle_api_ici",
        "FONCIERMCP_URL": "https://app-production-71c1.up.railway.app/mcp"
      }
    }
  }
}
4
Redémarrez Claude Desktop
Un symbole 🔌 apparaît en bas de la fenêtre — FoncierMCP est actif. Tapez "Analyse le 15 rue Garibaldi, Lyon 3e".

🔑 Authentification

Chaque requête inclut la clé API dans l'en-tête X-API-Key.

bash
curl -X POST https://app-production-71c1.up.railway.app/api/demo \
  -H "Content-Type: application/json" \
  -d '{"address": "15 rue Garibaldi, 69003 Lyon"}'

L'endpoint /api/demo est gratuit et sans auth (limité à 10 req/h par IP).

🛠 Référence des outils

analyze_parcelOUTIL PRINCIPAL
Analyse complète : cadastre + DVF + PLU + géorisques + DPE en parallèle. Temps moyen : 1,8 secondes.
ParamètreTypeRequisDescription
addressstringrequisAdresse complète (numéro, rue, code postal, ville)
dvf_radiusintegeroptionnelRayon DVF en mètres (défaut : 500, max : 2000)
get_dvf_history
Historique transactions DVF autour d'une adresse. Source : DGFiP / data.gouv.fr.
ParamètreTypeRequis
addressstringrequis
radiusintegeroptionnel
check_plu_rules
Zone PLU et constructibilité. Source : Géoportail Urbanisme IGN/DGALN.
get_erp_risks
Risques naturels et technologiques (ERP). Source : BRGM / Géorisques.
get_dpe
Distribution étiquettes DPE dans la commune. Source : Base nationale DPE ADEME.
